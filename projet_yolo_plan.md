# Projet 3 — Analyse d’images avec YOLO (You Only Look Once)

## 1. Sujet du projet
**Analyse d’images avec YOLO (You Only Look Once) — Détection d’objets en temps réel assistée par l’IA**

Nous allons explorer la technologie YOLO (modèle de vision par ordinateur de type “object detection”), une technologie émergente permettant d’identifier des objets dans une image ou une vidéo (personnes, animaux, voitures, etc.) en quelques millisecondes.

---

## 2. Technologies explorées

### Technologie principale
- **YOLOv8 (Ultralytics)** : modèle de détection d’objets moderne et rapide.

### Technologies complémentaires
- Python  
- Bibliothèques : `ultralytics`, `OpenCV`, `torch`  
- Environnement : Jupyter Notebook ou script Python  
- (Optionnel) Déploiement : Docker, FastAPI, interface web simple

### IA utilisées dans le projet
- ChatGPT  
- GitHub Copilot  
- Gemini ou Claude

---

## 3. Objectifs du projet
- Comprendre le fonctionnement d’un modèle de vision artificielle moderne.
- Comparer YOLO à d’autres options.
- Créer un prototype fonctionnel (image + vidéo + webcam).
- Utiliser l’IA pour générer et corriger du code.
- Rédiger un rapport + produire une démonstration de 10 minutes.

---

## 4. Plan d’exploration

### Semaine 1 – Choix et planification
- Validation du sujet YOLO  
- Création du dépôt GitHub  
- Rédaction du plan  
- Compréhension des bases YOLO

### Semaine 2 – Exploration technique
- Lire la documentation YOLOv8  
- Tester YOLO sur quelques images  
- Comprendre bounding boxes, scores de confiance  
- Utiliser l’IA pour clarifier les concepts

### Semaine 3 – Mise en pratique
- Intégrer YOLO dans un script Python  
- Détection d’objets dans une image + vidéo + webcam  
- Débogage assisté par IA

### Semaine 4 – Prototype final
- Application Python complète  
- Ou API + interface simple  
- Documentation assistée par IA

### Semaine 5 – Rapport + Démo
- Rédaction du rapport synthèse  
- Journal de bord  
- Préparation de la démo 10 minutes  
- Vérification GitHub

---

## 5. Livrables
- Dépôt GitHub complet  
- Prototype YOLO fonctionnel  
- Démonstration (10 minutes)  
- Rapport synthèse (3 à 5 pages)

---

## 6. Utilisation de l’IA dans le projet
Nous allons utiliser l’IA pour :  
- Comparer les technologies  
- Résumer la documentation  
- Générer du code YOLO  
- Corriger des erreurs Python  
- Optimiser le prototype  
- Rédiger documentation + rapport

Tous les prompts importants seront enregistrés dans le journal de bord.
